# 年会抽奖工具使用说明 - Pro版本

## 📋 工具简介

这是一个专业版的年会抽奖工具，拥有更震撼的视觉效果和更简洁的操作体验！完全基于浏览器运行，无需安装任何软件，方便在不同电脑上使用。

## ✨ 主要功能

1. **自定义Logo** - 可上传公司Logo显示在右上角，支持大小调节和重置
2. **自定义标题** - 年会标题可以自由设置
3. **自定义背景** - 可上传背景图片，支持重置为默认渐变背景
4. **侧边栏设置** - 所有设置隐藏在侧边栏，主界面更清爽
5. **全屏抽奖界面** - 放大的抽奖展示，更震撼
6. **名单导入** - 支持 CSV、Excel 格式的抽奖名单导入
7. **显示设置** - 可自由选择名单中要显示的列
8. **奖品设置** - 灵活的奖品等级、自定义名称和中奖人数设置
9. **抽奖功能** - 一键开始/停止抽奖，公平随机
10. **历史记录** - 完整的抽奖历史记录，方便查看
11. **响应式设计** - 适配不同尺寸的电脑屏幕
12. **美观效果** - 金色主题、超大彩带效果、动态展示

## 🎨 界面说明

### 主界面
- 左上角：菜单按钮，点击打开设置侧边栏
- 右上角：自定义Logo（上传后显示）
- 顶部：自定义年会标题
- 中间：超大抽奖展示区域
- 底部：开始/停止/重置按钮

### 侧边栏设置
- 标题设置：自定义年会标题
- Logo上传：上传公司Logo，调节大小，可重置
- 背景图片：上传背景图，可重置为默认背景
- 导入名单：上传抽奖名单
- 显示设置：根据上传的名单选择要显示的列
- 奖品设置：选择奖品等级、自定义奖品名称和中奖人数
- 历史记录：查看所有中奖记录

## 🚀 使用步骤

### 1. 打开工具

直接双击 `lottery-pro.html` 文件，即可在浏览器中打开抽奖工具。

### 2. 打开设置侧边栏

**方法一：点击菜单按钮**
- 点击左上角的菜单按钮（三条横线）

**方法二：关闭侧边栏**
- 点击抽奖主界面的任意空白区域
- 或者再次点击菜单按钮

### 3. 上传Logo（可选）

1. 在"Logo上传"区域点击或拖拽Logo图片
2. 支持PNG（推荐）、JPG、GIF、WebP格式
3. 上传后会显示在右上角
4. 使用滑块调节Logo大小（50%-200%）
5. 如需移除Logo，点击"重置Logo"按钮

### 4. 设置背景（可选）

1. 在"背景图片"区域点击或拖拽背景图片
2. 支持PNG、JPG、GIF、WebP格式
3. 上传后自动替换默认红色渐变背景
4. 如需恢复默认背景，点击"重置背景"按钮

### 5. 设置年会标题

在"标题设置"区域输入您的年会标题，主界面会实时更新显示。

### 6. 导入抽奖名单

**方法一：点击上传**
- 在侧边栏点击"导入抽奖名单"区域
- 选择您的抽奖名单文件（CSV 或 Excel 格式）

**方法二：拖拽上传**
- 直接将文件拖拽到上传区域

**名单格式要求：**
文件可包含任意列，系统会自动识别：
- 姓名（必需）
- 工号/ID（推荐）
- 部门（可选）
- 电话（可选）
- 其他自定义字段

示例格式：
```
工号,姓名,部门,电话
001,张三,技术部,13800138001
002,李四,市场部,13800138002
```

### 7. 显示设置

上传名单后，在"显示设置"区域：
- 勾选您想要显示的列
- 默认勾选姓名和工号列
- 可随时更改显示设置

### 8. 设置奖品

1. 在"奖品设置"区域选择奖品等级
   - 特等奖
   - 一等奖
   - 二等奖
   - 三等奖
   - 四等奖
   - 五等奖
   - 阳光普照奖
2. 设置自定义奖品名称（可选）
   - 例如："iPhone 15 Pro Max"、"华为Mate 60"
3. 设置中奖人数
4. 开始抽奖前确认设置
5. 可以关闭侧边栏，只保留清爽的主抽奖界面

### 9. 开始抽奖

1. 点击"开始抽奖"按钮
2. 屏幕会快速滚动显示候选人
3. 剩余人数≥10人时显示10个卡片同时滚动
4. 剩余人数<10人时显示1个卡片滚动
5. 点击"停止"按钮确定中奖者
6. 中奖者会以大卡片形式展示，并记录到历史中
7. 同时会有超多彩带效果！

### 10. 重置抽奖

如需重新开始抽奖，点击"重置"按钮即可。

## 🎯 奖品等级

- 特等奖（默认1人）
- 一等奖（默认1人）
- 二等奖（默认2人）
- 三等奖（默认3人）

- 五等奖（默认10人）

工具支持以下屏幕尺寸：

- 中等屏幕（900-1200px）：自适应展示

- 手机端：页面可滚动，操作流畅

## 🔧 常见问题

**Q: 如何打开设置面板？**
A: 点击左上角的菜单按钮（三条横线）即可打开侧边栏。

**Q: 如何快速关闭设置面板？**
A: 点击主界面的任意空白区域即可关闭侧边栏。

**Q: 如何上传Logo？**
A: 在侧边栏的Logo上传区域点击或拖拽图片文件即可。

**Q: Logo尺寸不合适怎么办？**
A: 使用上传后出现的滑块调节Logo大小，范围50%-200%。

**Q: 如何移除Logo？**
A: 点击"重置Logo"按钮即可隐藏Logo。

**Q: 如何上传背景图片？**
A: 在侧边栏的背景图片上传区域点击或拖拽图片文件即可。

**Q: 背景图片不满意怎么办？**
A: 点击"重置背景"按钮恢复默认红色渐变背景。

**Q: 显示设置如何使用？**
A: 上传名单后，勾选您想要在抽奖中显示的列即可。

**Q: 如何保存抽奖结果？**
A: 抽奖历史会显示在侧边栏，您可以截图保存或复制记录。

**Q: 能否中途更改奖品设置？**
A: 可以，在未开始抽奖时随时可以在侧边栏更改奖品等级、名称和人数。

**Q: 工具会联网吗？**
A: 不会，所有数据都在本地处理，保护您的隐私安全。

**Q: 如何恢复已中奖人员？**
A: 点击"重置"按钮，所有人员会恢复到待抽奖状态。

**Q: 标题可以随时修改吗？**
A: 是的，在侧边栏的标题设置中随时可以修改，主界面会实时更新。

**Q: 自定义奖品名称有什么用？**
A: 设置后，抽奖时主界面会显示您设置的奖品名称（如"iPhone 15"），更直观。

**Q: 名单必须包含哪些列？**
A: 至少需要包含姓名列，其他列是可选的。上传后可以在显示设置中选择要展示的列。

**Q: 滚动时第一个卡片不滚动怎么办？**
A: 现在已修复此问题，所有卡片都会正常同步滚动。
